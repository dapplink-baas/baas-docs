# DappLink BaaS 技术文档

## 一、概述 Overview

在 Web3 快速发展的时代，开发者与企业在构建去中心化应用（dApps）时面临着三个共性难题：

1.  **资产安全托管难** ------单点私钥风险大、黑客攻击频发、企业级安全门槛高
2.  **可信数据源缺失** ------区块链无法直接访问链下世界，数据真实性难以保证
3.  **应用创新门槛高** ------特别是预测市场、金融衍生品等复杂场景，需要更底层的专用基础设施支持

DappLink BaaS（Blockchain as a Service）正是为解决上述痛点而设计的

**去中心化基础设施平台**。平台由三大核心模块构成：

-   **MPC 托管系统（安全资产托管）**
-   **DappLink Oracle（可信数据服务）**
-   **事件预测平台（Prediction Engine，创新应用范式）**

DappLink BaaS 的使命，是成为"**连接传统金融与 Web3
世界的技术桥梁**"，帮助开发者以更低门槛、更高安全性、更强扩展性构建下一代去中心化应用。

# 三、MPC 托管系统（资产安全托管）

## 1. MPC（多方安全计算）是什么？

MPC Custody
是一种**去中心化密钥管理与签名技术**，通过将原始私钥拆分为多个密钥分片（Share），分别存储在不同节点。

特性：

-   私钥从未完整出现\
-   无须信任单一托管方\
-   阈值签名完成交易\
-   支持企业级高频交易场景

## 2. 技术优势

### 密钥永不落地

密钥在任何阶段都不会以完整形式出现，避免被盗风险。

### 阈值签名机制

以 3/5、5/9、7/11 等模式运行：

-   提升容错性\
-   无单点故障\
-   更高的安全级别

### 实时交易签名

不同于传统多签（人工确认慢），MPC 支持后台实时交易执行：

-   交易所热钱包\
-   高速清结算\
-   链游戏奖励发放\
-   DAO 财库自动管理

### 支持 TEE / CloudHSM

满足对安全等级要求极高的机构，例如交易所、合规资产托管方。

## 3. 安全机制设计

MPC 系统采用多层安全体系：

### 分布式节点网络

-   多区域部署\
-   多云架构\
-   专有节点隔离

即使攻击部分节点，也无法获得完整密钥。

### 多层加密通信

-   TLS 通道\
-   会话密钥（Session Key）\
-   零知识证明（ZK）可选

确保签名过程不可监听与篡改。

### 无状态签名（防重放攻击）

每次签名都有唯一 session ID。

### 审计追踪系统

记录：

-   谁发起交易\
-   哪些节点参与签名\
-   签名耗时\
-   异常阈值告警

## 4. 支持的应用场景

-   加密交易所热钱包托管\
-   Web3 DApp 后端资产管理\
-   游戏经济资产结算系统\
-   DAO 预算与国库\
-   项目方资金托管\
-   跨链桥验证节点

# 四、DappLink Oracle（可信数据服务）

## 1. 为什么需要 Oracle？

区块链无法直接读取：

-   金融市场价格\
-   天气数据\
-   体育赛事结果\
-   Web2 API\
-   随机数

Oracle 成为"链外世界 → 区块链"的数据桥梁。

## 2. Oracle 的核心服务

DappLink Oracle 提供三大基础能力。

### （1）Price Feed（价格预言机）

用途：

-   DeFi 借贷和清算系统\
-   DEX 报价\
-   衍生品资产定价\
-   NFT 地板价评估

技术特性：

-   多数据源聚合\
-   异常值防操纵算法\
-   去中心化节点共识\
-   秒级更新频率

### （2）VRF（可验证随机数）

应用：

-   Web3 游戏\
-   抽奖/盲盒\
-   NFT 随机属性生成\
-   预测市场随机抽样

特点：

-   不可预测\
-   不可操纵\
-   链上可验证

### （3）Event Prediction（事件结果判定）

为预测市场提供可验证的事件结果：

-   体育比赛\
-   金融价格判断\
-   娱乐事件（票房、开播等）\
-   政治事件（选举结果）\
-   商业事件（融资是否成功）

技术点：

-   共识化结果判定\
-   冲突仲裁\
-   结果签名 + 链上验证

## 3. Oracle 技术架构

    采集层 → 验证层 → 共识层 → 链上发布层
                     ↓
                  MPC 安全签名
